# 루틴 앱 배포 가이드

## 파일 구조
```
my-routine/
├── index.html       ← 웹앱 본체
├── sw.js            ← 서비스워커 (알림 처리)
├── manifest.json    ← PWA 설정
├── vercel.json      ← 배포 설정
└── icons/
    ├── icon-96.png
    ├── icon-192.png
    └── icon-512.png
```

---

## Step 1 — GitHub에 올리기

1. [github.com](https://github.com) 접속 → 로그인 (계정 없으면 무료 가입)
2. 우측 상단 **+** → **New repository**
3. Repository name: `my-routine`
4. **Public** 선택 → **Create repository**
5. 로컬에서 업로드:
   - `my-routine` 폴더를 통째로 드래그해서 GitHub 페이지에 드롭
   - 또는 GitHub Desktop 앱 사용 (코드 없이 가능)

---

## Step 2 — Vercel로 배포하기

1. [vercel.com](https://vercel.com) 접속 → **Sign up with GitHub**
2. **Add New Project** → 방금 만든 `my-routine` 저장소 선택
3. 설정 그대로 두고 **Deploy** 클릭
4. 1분 뒤 `https://my-routine-xxx.vercel.app` 형태의 URL 생성됨

---

## Step 3 — 아이폰 홈 화면에 추가 (PWA)

1. 아이폰 **사파리**로 위 Vercel URL 접속
   - ⚠️ 크롬/카카오 내 브라우저는 안 됨 — 반드시 사파리로!
2. 하단 공유 버튼(📤) 탭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **추가**
5. 홈 화면에 초록색 아이콘이 생김

---

## Step 4 — 알림 허용

1. 홈 화면 아이콘으로 앱 열기 (사파리가 아닌 아이콘으로!)
2. 앱 하단 **"알림 허용하기"** 버튼 탭
3. 아침/저녁 시간 설정
4. iOS 알림 허용 팝업 → **허용** 탭

완료! 🎉 이제 매일 아침·저녁 잠금화면에 알림이 와요.

---

## 자주 묻는 것

**알림이 안 와요**
- 설정 → 알림 → 해당 앱 → 알림 허용 확인
- 반드시 홈 화면 아이콘으로 열어야 함 (사파리 탭으로 열면 알림 비활성)

**데이터는 어디에 저장돼요?**
- 내 아이폰 로컬 저장소(localStorage)에만 저장됩니다.
- 외부 서버로 전송되지 않아요.

**다른 기기에서도 쓰고 싶어요**
- 각 기기에서 같은 URL로 접속 후 홈 화면 추가하면 됨
- 단, 체크 기록은 기기별로 따로 저장됩니다
